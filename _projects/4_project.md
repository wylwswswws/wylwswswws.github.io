## Overview

As a **Data Analyst** at **Novo Nordisk (China)**, I led two major initiatives: a company-wide database migration in collaboration with Microsoft China, and a predictive analytics project that optimized pharmaceutical sales promotion strategies.

## Database Migration

Collaborated with Microsoft China to lead a company-wide SQL database migration, consolidating **five disparate databases** into a unified system.

- Conducted thorough data flow analysis across all source systems
- Optimized SQL queries, **reducing runtime by 30%** (from 4.2s to 2.8s average)
- Documented migration processes and provided cross-functional support for smooth adoption across 200+ users

## Predictive Sales Analytics

Leveraged Python and statistical techniques to process and analyze historical sales promotion data for pharmaceuticals.

- Built predictive models with **SHAP interpretability** to understand which promotional factors drove the highest ROI
- Generated insights that **improved promotional budget allocation accuracy by 20%**
- Contributed to **$100,000 in cost savings** through data-driven budget optimization
- Delivered findings via **Power BI** dashboards with DAX-powered metrics

## Tech Stack

`SQL` · `Python` · `Power BI` · `DAX` · `SHAP` · `Statistical Modeling`
